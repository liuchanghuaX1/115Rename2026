# 115Rename2026

> 版本：**1.7.0**

专为 115 网盘设计的视频整理与重命名脚本，本地加工与多站刮削并行，智能缓存、并发、分段规范化，集评分、归档于一体。

## ✨ 功能

### 🧹 本地番号加工
- 番号识别（`T28`、`S2M`，`h637`→`TokyoHot-h637`）
- 分段统一（`part1`、`.Part.2`、`(CD3)` → `-1`、`-2`、`-A`）
- 智能标记：`4K`/`8K`/`60fps`/`120fps`/`VR`/`中文字幕`/`无码`/`流出`
- 广告域名、乱码自动清除

### 🌐 多站改名
- 轮询 `JavLibrary → JavBus → xslist → JavDB`
- 补全标题、女优、日期，生成规范名称
- 同名番号全局缓存

### 📁 归档
- **按女优**：自动创建女优文件夹并移动
- **按番号系列**：按系列前缀归档（如 `SSIS`）
- **女优+系列**：复合文件夹
- 需先设置归档根目录

### ⭐ 评分
从 JavDB 抓取评分写入 115 星级

### ⚡ 效率
并发队列（本地5/联网3/归档3/评分2），进度条

## 📋 使用

1. 登录 115，选中文件/文件夹（可多选）
2. 在 **更多** 菜单选择功能
3. 查看进度条，等待完成

## ⚠️ 限制

- FC2 不支持在线改名（仅本地加工）
- 需保持 `@connect` 权限

## 🚀 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 从 [GreasyFork](https://greasyfork.org/zh-CN/scripts/574424-115rename2026) 安装或手动导入
