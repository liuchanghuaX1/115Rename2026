# 115Rename2026

> 版本：**1.8.6**

一款强大的 115 网盘视频整理与重建名脚本，支持本地加工与多站刮削双轨并行，长厂牌优先匹配，杜绝误伤前缀；新增文件名备份与改名前后对比导出功能。

## ✨ 主要功能

### 🧹 本地番号加工
- 精准番号提取（`T28`、`S2M`、`MXVR`、`SIVR` 等长厂牌绝对优先）
- 分段智能统一（`.Part.2` → `-2`）
- 独立标记识别（`4K`、`VR`、`无码`、`中文字幕` 等），前缀内部的 `VR` 不被破坏
- 域名、广告残留自动清理

### 🌐 多站改名（在线刮削）
- 四引擎轮询：**JavLibrary → JavBus → xslist → JavDB**
- 自动补全正式标题、女优、日期
- 全局缓存，避免重复请求

### 📁 归档整理
- 按女优 / 按番号系列 / 女优+系列三种方式自动分类
- 支持设置归档根目录

### ⭐ JavDB 评分同步
- 从 JavDB 获取评分，写入 115 文件星级

### 💾 文件备份与对比
- **备份文件名**：选中文件后可以导出文件名列表为 TXT 或复制到剪贴板
- **改名对比**：每次改名完成后可导出「原文件名 → 新文件名」对照表

### ⚡ 高效并发
- 并发控制（本地 5 / 联网 3 / 归档 3 / 评分 2）
- 可视化进度条

## 📋 使用方法
1. 登录 115 网盘，选中文件或文件夹（可多选）
2. 在 **更多** 菜单中点击对应功能按钮
3. 根据提示操作，处理完成后可查看进度条与通知

## 🚀 安装
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 直接点击脚本的原始文件链接安装（或复制脚本内容手动粘贴）
3. 确保在 Tampermonkey 中已允许脚本访问 `javbus.com`、`javlibrary.com`、`xslist.org`、`javdb.com` 等域名
