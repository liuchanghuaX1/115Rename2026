# 115Rename2026

这是一款专为 115 网盘设计的视频整理与重命名增强脚本。采用“本地番号加工”与“网络刮削改名”双轨分离架构，提供最纯净、最精准的影片命名与归档体验。

## ✨ 核心功能与特色

- **双轨处理逻辑**：
  - **本地番号加工**：纯净提取番号、分集片段（如 `-1`、`-A`、`-C`）与高清/破解标记（如 `4K`、`无码`），废除原标题中的括号乱码，日期自动挂载于末尾。
  - **网络统一改名**：利用纯净番号去 JavBus 进行网络刮削，获取正规影片名与女优名，彻底抛弃本地乱码标题。
- **强大抗干扰引擎**：绝对免疫广告域名前缀（如 `www.xxx.la@`），并精准剔除与番号沾连的日期（支持识别 `2014 01 01` 等带空格格式）。
- **根治重复番号与恶意前缀**：自动识别并清除 `01SIRO-1736` 等干扰前缀，确保文件名中番号只在最前方出现一次。
- **厂商格式规范化**：针对 `TokyoHot` (单字母番号如 `N1032`)、`1pondo` 等特殊厂商进行标准化重命名。
- **UI 稳定兼容**：底层 UI 注入采用最稳定兼容的架构，不盲目追求新版，确保最高可用性。

## ⚠️ 已知问题 (v1.0.0)

- 暂不支持通过 **JavDB** 进行数据刮削。
- 暂不支持 **FC2** 网站的专属番号改名与刮削。
- *(以上功能将在后续版本中逐步探索与完善)*

## ⚙️ 统一命名格式

无论使用哪种方式，最终都会将杂乱的文件名整理为统一标准：

- **网络改名**：`[番号(含分集)] [刮削影片名] [刮削女优名] [【保留标记】] [_YYYY-MM-DD] [后缀]`
- **本地加工**：`[番号(含分集)] [清洗后的剩余本地标题] [【保留标记】] [_YYYY-MM-DD] [后缀]，只修改本地文件名，不需要联网`

3. 也可以前往 [SleazyFork 页面](https://sleazyfork.org/) 搜索 "115Rename2026" 进行安装。
