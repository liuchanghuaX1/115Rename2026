# 115Rename2026

> 版本：**1.7.2**

为 115 网盘设计的视频整理与标准化重命名脚本，本地加工与多站刮削并行，支持分段统一、智能标记、广告清理、评分同步及归档整理。

## ✨ 功能

### 🧹 本地番号加工
- 番号提取与标准化（`T28`、`S2M` 等混合厂牌；`h637` → `TokyoHot-h637`）
- 分段智能处理：`part1`、`.Part.2`、`(CD3)` → 统一 `-1`、`-2`、`-A`
- 标记自动识别：`4K`/`8K`/`60fps`/`120fps`/`VR`/`中文字幕`/`无码`/`流出`
- 广告残留清理：自动移除 `[3Q]`、`(原)` 等垃圾标牌及视频编码信息

### 🌐 多站改名（在线刮削）
- 四引擎轮询：**JavLibrary → JavBus → xslist → JavDB**
- 补全正式标题、女优、发行日期，生成 `番号 标题 女优名 【标记】_日期.ext`
- 同名番号全局缓存，避免重复请求

### 📁 归档整理
- 按女优 / 按番号系列 / 女优+系列三种归整方式
- 自动创建文件夹并移动文件，支持设置归档根目录

### ⭐ JavDB 评分
- 从 JavDB 获取评分（5分制），自动写入 115 文件星级

### ⚡ 高效并发
- 本地加工并发 5，联网改名并发 3，归档并发 3，评分并发 2
- 右下角实时进度条，处理结果实时通知

## 📋 使用指南

1. 登录 115 网盘，选中文件或文件夹（可多选）
2. 点击 **更多** 菜单，选择对应功能：
   - **本地番号加工**：仅用文件名信息标准化（无需联网）
   - **改名(多网站轮询)**：在线补全完整信息
   - **归档至文件夹**：选择归整模式（需先设置归档根目录）
   - **设为归档根目录**：右键目标文件夹设置
   - **获取javdb评分**：批量更新星级评分
3. 处理完成会有通知，可在右下角查看进度

## 🚀 安装

### 从 GitHub 安装（推荐，支持自动更新）
1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 访问脚本原始文件链接，Tampermonkey 会自动识别并提示安装：  
   [https://raw.githubusercontent.com/liuchanghuaX1/115Rename2026/main/115Rename2026.user.js](https://raw.githubusercontent.com/liuchanghuaX1/115Rename2026/main/115Rename2026.user.js)
3. 点击 **安装** 即可，后续版本更新会自动推送。

### 从 GreasyFork 安装（备用）
前往 [115Rename2026 脚本主页](https://greasyfork.org/zh-CN/scripts/574424-115rename2026) 点击安装（发布后更新至最新版本）。

## ⚠️ 注意
- FC2 番号暂不支持在线信息查询（会自动降级为本地加工）
- 请确保 Tampermonkey 中已允许脚本访问 `javbus.com`、`javlibrary.com`、`xslist.org`、`javdb.com` 等域（`@connect` 权限）
