# 115Rename2026

> 版本：**1.9.1**

为 115 网盘设计的视频整理与重命名脚本，支持本地加工、多站刮削、归档、评分及备份导出。分段永久保留，长厂牌优先，杜绝误伤。

## ✨ 功能

- **本地番号加工**  
  精准提取番号（长厂牌 `MXVR`、`SIVR` 等优先），分段统一 (`Part.2` → `-2`，`-3` 直接识别)，标记智能聚合 (`4K`、`VR`、`无码`、`中文字幕` 等)。域名、广告残留自动清除。

- **多站改名**  
  轮询 JavLibrary → JavBus → xslist → JavDB，补全标题、女优、日期。番号基础部分不变，分段自动附加。

- **归档整理**  
  按女优 / 按番号系列 / 女优+系列三种方式自动创建文件夹并移动文件，支持设置归档根目录。

- **JavDB 评分**  
  抓取 JavDB 评分并写入 115 文件星级。

- **备份与对比**  
  可导出选中文件名（TXT 或剪贴板）；改名完成后可导出新旧对照表。

- **高效并发**  
  本地加工 5 并发，联网改名 3 并发，归档 3 并发，评分 2 并发，带进度条。

## 📋 用法

1. 登录 115 网盘，选中文件或文件夹（可多选）。
2. 点击 **更多** 菜单，选择对应功能按钮。
3. 根据提示操作，任务完成会收到通知，可选导出对比结果。

## 🚀 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展。
2. 新建脚本，粘贴完整代码，保存并启用。
3. 确保 Tampermonkey 已允许脚本访问 `javbus.com`、`javlibrary.com`、`xslist.org`、`javdb.com` 等域名（`@connect` 权限）。
