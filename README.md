# 115Rename2026

一款专为 115 网盘设计的视频整理与标准化重命名脚本。采用“本地加工”与“网络刮削”双轨分离架构。

## ✨ 主要功能
- **本地番号加工**：提取番号、分段（part1 转 -1）与日期，保留本地中文标题，实现文件名标准化。
- **厂牌深度支持**：完美支持 `T28`、`S2M` 等数字混合厂牌，以及 `h637` 等单字母番号（自动规范为 TokyoHot）。
- **防误伤引擎**：采用透视匹配算法，解决 `DASS` 等前缀被错误截断的顽疾。
- **纯净体验**：自动根治 `01` 前缀、广告域名、重复番号及 `[3Q]` 等乱码括号。

## ⚠️ 已知问题
- 暂不支持 JavDB 数据刮削。
- 暂不支持 FC2 网站的专属改名。

## 🚀 安装方法
1. 首先安装用户脚本管理器扩展（如 [Tampermonkey](https://www.tampermonkey.net/)）。
2. 前往 GreasyFork 页面安装：[115Rename2026 脚本主页](https://greasyfork.org/zh-CN/scripts/574424-115rename2026)。
